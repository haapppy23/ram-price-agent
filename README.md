# RAM Price Agent

국내 PC RAM 시세를 조회하고, 검색 결과 중 실제 RAM 상품만 선별한 뒤 AI가 TOP 5를 추천하는 단일 목적 에이전트입니다.

---

## 1. 프로젝트 개요

RAM Price Agent는 사용자가 입력한 RAM 조건을 기준으로 국내 쇼핑 검색 결과를 수집하고,  
그중 실제 RAM 상품만 골라낸 뒤 가격을 정리하고 TOP 5 추천을 제공하는 웹 기반 프로젝트입니다.

이 프로젝트는 범용 챗봇이 아니라, **RAM 시세 조회와 추천만 수행하는 단일 목적 에이전트**입니다.

---

## 2. 프로젝트 목적

사용자가 원하는 RAM 조건을 입력하면:

- 관련 상품 검색
- 실제 RAM 상품만 선별
- 가격 정보 요약
- TOP 5 추천

까지 한 번에 제공하는 것을 목표로 합니다.

즉, 단순 검색 결과 나열이 아니라  
**검색 → 필터링 → 요약 → 추천** 흐름을 하나의 목적에 맞게 수행합니다.

---

## 3. 핵심 기능

### 3-1. RAM 조건 입력
사용자는 아래 조건을 입력할 수 있습니다.

- DDR4 / DDR5
- 16GB / 32GB
- 클럭 값 입력 가능  
  예: 3200, 5600, 6000

### 3-2. 검색 결과 수집
Selenium을 사용하여 검색 결과 페이지를 렌더링한 뒤 상품 후보를 수집합니다.

### 3-3. AI 기반 상품 선별
Ollama 로컬 모델을 이용해 검색 결과 중에서 실제 데스크탑 RAM 상품만 선별합니다.

제외 대상 예시:
- 노트북용 RAM
- SO-DIMM / SODIMM
- 조립PC / 완본체 / 풀세트
- CPU / 본체 / 세트 상품
- 조건과 맞지 않는 상품

### 3-4. 가격 요약
선별된 상품을 기준으로 다음 정보를 계산합니다.

- 최저가
- 평균가
- 최고가

### 3-5. AI TOP 5 추천
선별된 RAM 상품 중에서 AI가 조건 적합성, 가격, 브랜드 균형 등을 바탕으로 TOP 5를 추천합니다.
